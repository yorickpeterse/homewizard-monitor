# HomeWizard Monitor

A simple [Inko](https://inko-lang.org/) program for monitoring the output of a
[HomeWizard P1 meter](https://www.homewizard.com/), and sending it to an
InfluxDB/VictoriaMetrics database.

# Requirements

- Inko `main`
